# Red Trust - Full Stack Real Estate Application

A complete full-stack web application for Red Trust Real Estate with a landing page and admin panel.

## Features

### Landing Page
- Responsive design with modern UI
- Display projects from database with images, names, and descriptions
- Show client testimonials with images, names, and designations
- Contact form submission
- Newsletter subscription

### Admin Panel
- Project management (Add/Delete projects)
- Client management (Add/Delete clients)
- View contact form submissions
- View newsletter subscribers

## Tech Stack

### Backend
- Node.js with Express
- MongoDB with Mongoose
- Multer for file uploads
- CORS for cross-origin requests

### Frontend
- HTML5
- CSS3 with Flexbox/Grid
- Vanilla JavaScript
- Responsive design
